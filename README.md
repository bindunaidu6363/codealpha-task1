# CodeAlpha Disease Prediction Model üè•

 Project Overview

This project is developed as part of the **CodeAlpha Machine Learning Internship**.
The main objective of this project is to build a machine learning model that predicts whether a person is likely to have a disease based on medical attributes.

The model analyzes patient health data and classifies the result as **disease present** or **not present**.

---

Objective

* To analyze medical dataset
* To preprocess health data
* To build a classification model
* To predict disease probability accurately

---

 Dataset Used

**Dataset Name:** Pima Indians Diabetes Dataset

### Features in the dataset:

* Pregnancies
* Glucose Level
* Blood Pressure
* Skin Thickness
* Insulin Level
* BMI (Body Mass Index)
* Diabetes Pedigree Function
* Age

### Target Variable:

* **Outcome**

  * 0 ‚Üí No Disease
  * 1 ‚Üí Disease Present
